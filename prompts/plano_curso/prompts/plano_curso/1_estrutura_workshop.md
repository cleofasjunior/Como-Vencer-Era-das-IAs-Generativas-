# Prompt Utilizado para Estruturar o Workshop

**Ferramenta:** Claude 3.5 Sonnet / ChatGPT-4
**Objetivo:** Criar um roteiro de apresentação de alto impacto e curta duração.

## System Prompt
> "Você é um Designer Instrucional experiente em andragogia (educação para adultos). O seu objetivo é criar workshops curtos que combatam a resistência à tecnologia."

## User Prompt
> "Crie a estrutura de slides para uma oficina de 40 minutos chamada 'Criando sua Primeira Aula com IA'.
>
> **Restrições:**
> 1.  **Sem teoria chata:** Nada de explicar o que é Rede Neural. Vamos direto para o uso.
> 2.  **Foco na dor:** Comece abordando a falta de tempo do professor.
> 3.  **Modelo 'I do, We do, You do':** Eu mostro, nós fazemos juntos, eles fazem sozinhos.
> 4.  **Output:** Descreva o que deve ter em cada slide (texto e imagem sugerida) e o roteiro de fala para o apresentador.
>
> Quero que os professores saiam com a sensação de 'alívio' e não de 'mais trabalho'."

## Resultado e Ajustes
*A IA sugeriu inicialmente 1 hora de duração. Solicitei a redução para 40 minutos para caber no HTPC (Horário de Trabalho Pedagógico Coletivo) das escolas brasileiras.*
